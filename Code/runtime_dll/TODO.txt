

Runtime dll:


När C/C++ bibliotekskod ska användas borde det gå att i runtime kompilera det till .dll med Clang eller liknande C++-kompilationsbibliotek, och sen länka in det i mitt eget program. De funktionerna borde sen gå att köra i runtime.

Det kan också gå att använda c++ via System() och kompilera till .dll genom det.
